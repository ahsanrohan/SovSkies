# Sovereign Skies
## Description

Sovereign skies is a top down shooter. Take down waves of enemy planes while dodging incoming projectiles. 

## Controls:

| `key` | action|
|--|--|
| `1` | switch to plane 1 |
| `2` | switch to plane 2 |
|`left mouse` | shoot

## Installation
`pip install `
